# 📊 기업 재무 데이터 분석 프로젝트

## 1. 📌 분석 목적
기업 규모별 재무 데이터를 활용하여  
**성장성과 재무 리스크(부채비율)를 비교 분석**하고  
기업 유형별 특징을 파악하는 것을 목표로 한다.

---

## 2. 📁 데이터 설명
- 기간: 2020년 ~ 2024년  
- 주요 변수:
  - 기업규모
  - 업종명
  - 총자산
  - 부채
  - 자기자본  

※ 원본 데이터는 연도별 컬럼 형태(wide format)로 구성되어 있어  
분석을 위해 세로 구조(long format)로 재구성하였다.

---

## 3. 🛠 데이터 전처리

### ✔ 문제점
- 연도별 데이터가 컬럼으로 분리됨  
- 숫자 데이터가 TEXT 타입으로 저장됨  

### ✔ 해결 방법
- `CAST()`를 사용하여 숫자 변환  
- `UNION ALL`을 활용하여 연도 컬럼 생성

### 4-2. 기업 규모별 자산 성장률 비교

```sql
-- 기업 규모별 자산 성장률 비교
SELECT 
    기업규모,
    ROUND(
        (AVG(CASE WHEN 연도 = 2024 THEN 총자산 END) -
         AVG(CASE WHEN 연도 = 2023 THEN 총자산 END))
        / AVG(CASE WHEN 연도 = 2023 THEN 총자산 END)
    , 4) AS 성장률
FROM sample_data_long
GROUP BY 기업규모
ORDER BY 성장률 DESC;
```

```csv

한시성중소기업,0.3558
중견기업,0.2543
중기업,0.1018
대기업,0.0507
소기업,0.0503
```

👉 **해석**
- 한시성중소기업과 중견기업이 높은 성장률
- 대기업/소기업은 낮은 성장률 → 안정형 구조

### 4-3. 기업 규모별 재무 리스크 (부채비율)

```sql
-- 기업 규모별 재무 안정성 (부채비율)
SELECT 
    기업규모,
    연도,
    ROUND(AVG(부채 / 총자산), 4) AS 부채비율
FROM sample_data_long
GROUP BY 기업규모, 연도
ORDER BY 기업규모, 연도;
```

## 5. 🧠 최종 결론

본 분석에서는 기업 규모별 재무 데이터를 기반으로  
성장성과 재무 리스크 간의 관계를 비교하였다.

분석 결과, 기업 규모에 따라 성장 패턴과 재무 구조에  
뚜렷한 차이가 존재하는 것으로 나타났다.

- **한시성중소기업**은 가장 높은 성장률을 보였으나,  
  동시에 높은 부채비율을 유지하여 재무적 리스크가 큰 구조를 보였다.

- **중견기업**은 높은 성장률과 비교적 안정적인 부채비율을 동시에 나타내며  
  가장 균형 잡힌 재무 구조를 가진 것으로 분석되었다.

- **대기업**은 부채비율이 점진적으로 감소하며 안정성을 확보하고 있으나,  
  성장률은 상대적으로 낮아 성숙 단계에 진입한 것으로 해석된다.

- **소기업 및 중기업**은 완만한 성장세를 보이나  
  비교적 높은 부채비율을 유지하여 재무 부담이 지속되는 구조를 보인다.
