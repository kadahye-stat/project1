# 📊 기업 재무 데이터 분석 프로젝트

## 1. 📌 분석 목적
기업 규모별 재무 데이터를 활용하여  
**성장성과 재무 리스크(부채비율)를 비교 분석**하고  
기업 유형별 특징을 파악하는 것을 목표로 한다.

---

## 2. 📁 데이터 설명
- 기간: 2020년 ~ 2024년  
- 주요 변수:
  - 기업규모
  - 업종명
  - 총자산
  - 부채
  - 자기자본  

※ 원본 데이터는 연도별 컬럼 형태(wide format)로 구성되어 있어  
분석을 위해 세로 구조(long format)로 재구성하였다.

---

## 3. 🛠 데이터 전처리

### ✔ 문제점
- 연도별 데이터가 컬럼으로 분리됨  
- 숫자 데이터가 TEXT 타입으로 저장됨  

### ✔ 해결 방법
- `CAST()`를 사용하여 숫자 변환  
- `UNION ALL`을 활용하여 연도 컬럼 생성  

---

## 4. 📈 주요 분석 결과

### 4-1. 기업 규모별 자산 성장률

| 기업규모 | 성장률 |
|----------|--------|
| 한시성중소기업 | 35.58% |
| 중견기업 | 25.43% |
| 중기업 | 10.18% |
| 대기업 | 5.07% |
| 소기업 | 5.03% |

👉 **해석**
- 한시성중소기업과 중견기업이 높은 성장률을 보임  
- 대기업과 소기업은 상대적으로 낮은 성장률  

---

### 4-2. 재무 리스크 분석 (부채비율)

#### 📌 주요 특징
- **대기업**: 부채비율 감소 → 재무 안정성 개선  
- **소기업 / 중기업**: 높은 부채비율 유지 → 재무 부담 존재  
- **중견기업**: 비교적 안정적인 재무 구조  
- **한시성중소기업**: 높은 부채비율 유지 → 고위험 구조  

---

## 5. 🔍 핵심 인사이트
- **고성장 기업일수록 높은 재무 리스크를 동반**
- **중견기업은 성장성과 안정성을 동시에 확보한 구조**
- **대기업은 안정적이나 성장성은 제한적**

---

## 6. 🧠 결론
기업 규모에 따라  
**성장성과 재무 안정성 간의 트레이드오프 관계**가 존재한다.

특히, 한시성중소기업은 높은 성장률을 보였으나  
동시에 높은 부채비율을 유지하여  
재무적 지속 가능성 측면에서 주의가 필요하다.
